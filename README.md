# 🏆 تولید کننده تشویق‌نامه با تعداد بالا/ Certificate Generator


### معرفی
این ابزار به صورت خودکار تشویق‌نامه‌های PDF را برای دانش‌آموزان فعال در حوزه بهداشت تولید می‌کند. که مناسب برای معاونین بهداشت دانشگاه‌های علوم پزشکی و ادارات آموزش و پرورش می باشد.
(درصورت تمایل سازمانها میتوانند با تغییر فایل تشویقنامه از آن به عنوان صدور گواهی کارگاههای آموزشی نیز استفاده کنند.)

### ویژگی‌ها
- ⚡ خواندن خودکار از فایل اکسل
- 📝 جایگزینی نام و مدرسه در قالب Word
- 📄 خروجی PDF آماده چاپ
- 🖨️ کیفیت بالا مناسب برای پرینت
- 📁 مدیریت خودکار پوشه خروجی

### نیازمندی‌ها
- Python 3.7 یا بالاتر
- Microsoft Word (برای تبدیل به PDF)
- کتابخانه‌های پایتون (در کد نصب می‌شوند)

### نحوه استفاده

#### 1. ساختار فایل اکسل (winners.xlsx)
| NAME | SCHOOL |
|------|--------|
مثال:
| علی محمدی | شریعتی |
| فاطمه حسینی | طالقانی |

#### 2. ساختار قالب word(certificate.docx)
